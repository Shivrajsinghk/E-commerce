# E-Commerce Platform

A full-stack e-commerce web application built using React, Django REST Framework, and PostgreSQL.

## Live Demo
https://e-commerce-hy3v.vercel.app

## 🚀 Features

- User Authentication
- Product Listing
- Add to Cart
- Wishlist
- Responsive UI
- Admin Dashboard
- Order Management
- Search & Filtering

## 🛠️ Tech Stack

### Frontend
- React
- Tailwind CSS
- Axios

### Backend
- Django
- Django REST Framework
